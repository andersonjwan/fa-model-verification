Finite Automata Model Verification

The Finite Automata Model Verification tool enables users to compare their JFLAP designed Finite Automata against the instructor's verified solutions for accuracy and acceptance. Simply select an appropriate JFLAP file, upload, and compare against any of the verified selected solutions provided.

Select Finite Automata

Select the JFLAP file to upload that you would like to check.


Check Finite Automata

With the selected Finite Automata uploaded, select a solution to compare it against below and click "Check My Answer" below.
